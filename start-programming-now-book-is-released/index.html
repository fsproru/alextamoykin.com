Start Programming Now Book is Released

05 May 2014 on Book

What

I’m super excited to annouce that the Start Programming Now book is released. If you would like to get into coding but you don’t know where to start, this book is for you. And you can read if for FREE.

Why

Technology is an important part of our live and we need more and better developers. The entry barrier to get to software development is still high. That’s why I decided to write a book that helps non technical people to get the ball rolling with programming. It’s great to hear from readers that everything start making sense after reading the book. It has been an absolute pleasure to work together with every reader towards understanding the basics of programming.

How

You can read the book for free online. Additionally, you can thank me by purchasing it and read it offline on your laptop, phone, tablet or kindle. Enjoy!
